# To-Do App (Vanilla JavaScript)

A simple and interactive To-Do application built using HTML, CSS, and JavaScript.
*Current Version:* v1.0

## Features
- Create/Add tasks
- Read/View tasks
- Update/Edit tasks
- Delete tasks
- Mark tasks as completed
- Prevents Empty Task Submission
- Prevents Duplicate Task Submission
- Delete Confirmation
- Clean and responsive UI (coming soon)
  
## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
 
## Upcoming Features
- Responsive Design
- Local Storage
- Array of Objects Refactor
- Search Tasks
- Filter Completed Tasks
- Dark Mode
